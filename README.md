# PROJETO-COM-SERVO-WOKWI-
Neste projeto, que foi realizado na plataforma Wokwi, sendo ultilizado: 2 Pushbuttom, 2 LED's, 2 Resistor, 1 Servo, 1 SSD1306 OLED display. Funciona da seguinte forma: Quando o pushbuttom 1 (o azul) é pressionado, o Servo desce 90°, o led azul acende e no visor do SSD1306 exibe a mensagem: "luz acesa". E quando o pushbuttom 2 (o verde) é pressionado, o Servo volta a 0°, o led verde acende e no visor do SSD1306 exibe a mensagem: "luz apagada"
